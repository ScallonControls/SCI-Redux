Running external programs

This article was last updated 2021-09-17. 

Launching an external program from DeltaV Live

 

This procedure is for launching an external program from DeltaV Live. For files with the extension .ps1 or .bat please see <insert link to bat and ps1 conversion tool here>

Creating the standard

  1. Open DeltaV Graphics Studio from Start>DeltaV Engineering>Graphics Studio
  2. On the left sidepane in Library Explorer navigate to DeltaV_System/Library/Standards/Emerson/Applications
  3. Create a new Private String
    1. Rename PrivateString# to S_ApplicationName 
      (S_Notepad)
    2. For Value, input the applications full path along with ;NoSignatureCheck (C:\Windows\System32\notepad.exe;NoSignatureCheck)
  4. Publish your changes

 

Assigning OnClick action to a button

 

  1. In Graphics Studio on your Display, select your button.
  2. On the Right sidepane select the Interaction Tab
  3. Under the Button section, select the Action Open Application
  4. The DeltaV Appplication dropdown should be set to False
  5. Under the Applcation path dropdown you should see your newly created Standard. (This is S_Notepad in our example)]
  6. Save and Publish the graphic

 

Considerations

 

This will not account for the program opening multiple times. This will either need to be accomplished by the program itself or be handled with a powershell script converted into a an executable.
